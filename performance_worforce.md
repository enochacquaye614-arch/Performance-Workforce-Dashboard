# Performance-Workforce-Dashboard
An interactive HR Analytics Dashboard developed in Microsoft Excel to analyze employee performance, workforce demographics, salaries, tenure, and departmental insights. This project demonstrates how Excel can be used as a business intelligence tool to transform raw workforce data into actionable insights.

# Project Overview
Human Resource departments rely on workforce analytics to monitor employee performance, understand workforce composition, and support strategic decision-making.
This project analyzes employee data to identify workforce trends and visualize key HR metrics through an interactive dashboard.

# Business Problem
Managing workforce performance becomes increasingly challenging as organizations grow. HR managers require quick access to employee metrics to answer questions such as:
- Which departments have the highest-performing employees?
- What is the gender distribution across departments?
- Which regions have the largest workforce?
- How does tenure relate to performance?
- How are salaries distributed across salary bands?

# Objectives
- Clean and organize HR data
- Analyze workforce demographics
- Evaluate employee performance
- Explore salary distribution
- Compare departments and regions
- Build an interactive Excel dashboard
